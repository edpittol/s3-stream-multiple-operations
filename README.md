# s3-stream-multiple-operations

## Setup (Local PHP + Composer)

This project uses PHP with Composer for dependency management, running on a locally installed PHP (no Docker required).

### Prerequisites

- PHP 8.5 or higher
- Composer 2.0 or higher

### Installation

1. Install PHP and Composer on your machine if not already installed.
2. Clone this repository.
3. Navigate to the project directory and install dependencies:

```bash
composer install
```

This will download all required packages, including the AWS SDK for PHP, and generate the `vendor/` directory.
